# 人情记账本 / Gift Ledger v1.1.8

## v1.1.8 更新内容

- 人员管理新增出生日期。
- 出生日期支持选择：公历 / 农历。
- 新增 / 编辑记录时，如果事件类型是“生日”或英文包含 `birthday`，会自动根据人员出生日期和记录日期计算当前年龄。
- 生日年龄会保存到记录中，并在 PC 表格和移动端卡片的事件类型旁边显示。
- 修改人员出生日期或日期类型后，会自动刷新该人员已有生日记录的年龄。
- 兼容旧数据：旧人员没有出生日期时，不影响原有记录。
