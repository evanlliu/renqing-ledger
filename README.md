# 人情记账本 / Gift Ledger v1.2.6

## v1.2.6 更新内容

1. 移动端整体界面重新优化，主页面更紧凑、更直观，单页可显示更多信息。
2. 顶部统计卡片精简为 2 个：收入（净收入）和支出（净支出）。
3. 删除首页副标题“收人情、封人情、节日红包统一记录”。
4. 移动端筛选改为弹出层，点击弹窗外区域会直接关闭。
5. 修复“汇率设置”里的“获取当前汇率”按钮点击无效问题。
6. 优化所有配置页面的移动端拖动布局，修复拖动按钮导致的大面积空白行问题。
7. 移动端主页面记录卡片支持左滑显示操作按钮（历史 / 复制 / 编辑 / 删除）。
8. 配置页面继续支持左滑删除、拖动排序、删除确认。
9. 代码内版本号已同步为 v1.2.6。
