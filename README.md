# 人情记账本 / Gift Ledger v1.2.3

## v1.2.3 更新内容

1. 移动端配置删除增加二次确认弹窗。
2. 覆盖以下配置页面：
   - 人员管理
   - 关系配置
   - 分类配置
   - 类型管理
   - 红包规则
   - 汇率设置
3. 移动端仍然保持左滑显示删除按钮。
4. 点击删除后必须确认，避免误触。
5. PC 端删除配置时也会确认，避免误删。

## 更新说明

已有线上数据时，只更新：

```text
index.html
README.md
```

不要覆盖线上 `data.json`。

Cloudflare Worker 无变化，不需要更新 `worker.js`。
