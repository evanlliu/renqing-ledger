# 人情账 / Gift Ledger v1.2.11

## v1.2.11 更新内容

1. “X 条记录”移动到“记录列表”后面，同一行显示。
2. 移动端主页面左滑显示历史 / 复制 / 编辑 / 删除时，右下角悬浮按钮会自动隐藏；收起左滑后再显示。
3. 移动端记录卡片进一步压缩空白、缩小内边距和信息块高度，一页可以显示更多内容。
4. 移动端记录卡片之间的上下间距已缩小。
5. 第一次加载页面和点击同步按钮使用同一套逻辑：都会重新读取 GitHub 页面上的 data.json，再根据其中的 Cloudflare 配置读取云端数据，减少数据不同步问题。
6. 加载 / 同步提示统一为简短格式，例如：已加载 · 10:24:35、同步成功 · 10:24:35。
7. 代码内版本号已同步为 v1.2.11。
