# 人情记账本 / Gift Ledger v1.2.5

## v1.2.5 更新内容

1. 新增 / 编辑记录中的「货币」改为下拉选择。
2. 货币下拉会自动来自：默认货币、折算基准货币、汇率设置、已有记录货币、红包规则货币。
3. 汇率设置支持添加新货币时获取当前汇率。
4. 当前汇率通过 Frankfurter 接口获取；如果接口不可用，会继续使用手动输入或已有默认汇率。
5. 新增 / 编辑记录保存时，会按「记录日期 + 货币 + 折算基准货币」计算并保存该记录的折算汇率。
6. 统计折算优先使用记录自身保存的日期汇率；没有记录汇率时再使用汇率设置里的默认汇率。
7. 代码内版本号已同步为 v1.2.5。
